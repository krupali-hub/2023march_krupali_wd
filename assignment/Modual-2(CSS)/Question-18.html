Que:-18 What are the media types allowed by CSS?

Ans:-

  1. continuous or paged.
  2. visual, audio, speech, or tactile.
  3. grid (for character grid devices), or bitmap.
  4. interactive (for devices that allow user interaction), or static (for those that do not).
  5. all (includes all media types)
  6. screen: This media type is used for computer screens, tablets, smartphones, and other similar devices.
  7. tv: This media type is used for television-type devices.
  8. speech: This media type is used for speech synthesizers and screen readers that read the document.
