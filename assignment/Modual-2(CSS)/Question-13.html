Que:-13 What are the ways to integrate CSS as a web page?

Ans.-

There are three methods of adding CSS to a web page.

Inline css:-

1_ Inline css using the <style> attribute inside HTML elements

2_ An inline CSS is used to apply a unique style to a single HTML element.

exampal:-

A Blue Heading

A red paragraph.

Internal css:-

1_ An internal CSS is used to define a style for a single HTML page.

2_ An internal CSS is defined in the section of an HTML page, within a <style> element.

example:-

this is heading

this is paragraph

Extarnal css:-

1_ An external style sheet is used to define the style for many HTML pages.

2_ To use an external style sheet, add a link to it in the <head> section of each HTML page:
