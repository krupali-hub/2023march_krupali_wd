Que:-19 What is the rule set?

Ans:-

1_ The set of Rules agreed for a content,esp in martial arts.

2_ CSS (Cascading Style Sheets) has a some set of rules that define how web pages are styled and presented.

3_ The rule set in CSS of selectors and declarations

4_ Selectors: Selectors determine which elements on a web page the CSS rules should be applied to. There are various types of selectors in CSS, including.

5_ Element selectors: Select elements based on their tag name, such as p, h1, or div.

6_ Class Selector: Targets elements with a specific class attribute value. It is select by (.) followed by the class name.

7_ ID Selector: Targets an element with a specific ID attribute value. It is select by a hash symbol (#) followed by the ID name.
