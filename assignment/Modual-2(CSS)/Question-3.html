Qns:-3 What is the difference between CSS2 and CSS3?

Ans.:-

Difference between CSS2 and CSS3:-

1- CSS2 and CSS3 are different versions of the Cascading Style Sheets (CSS) language used for styling web pages.

2- CSS2 didn't have provision to specifically design the web layout.

3- CSS3 introduced new properties and values for styling backgrounds and borders.

4- CSS3 has compatibility with external font styles through google fonts and typecast. It was not possible with earlier CSS2.

5- The selectors in CSS3 has increased while CSS2 only had simple selectors.
