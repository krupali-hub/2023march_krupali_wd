Que.9 :-How is Cell Padding different from Cell Spacing?

Ans.:-

cell padding:-

_1 Cell padding is the space between the cell edges and the cell content.

_2 By default the padding is set to 0

_3 To add padding on table cells, use the CSS padding property:

Example:-

cellpadding

cellspacing:-

_1 Cell spacing is the space between each cell.

_2 By default the space is set to 2 pixels.

_3 To change the space between table cells, use the CSS border-spacing property on the table element

Example:-

cellspacing
