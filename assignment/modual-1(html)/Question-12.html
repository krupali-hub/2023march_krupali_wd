Que.12 How to create a Hyperlink in HTML?

Ans.:-

Hyperlink in Html:-

1.. HTML links are hyperlinks.

2.. You can click on a link and jump to another document.

3.. A link does not have to be text. A link can be an image or any other HTML element.

4.. An unvisited link is underlined and blue

5.. A visited link is underlined and purple

6.. An active link is underlined and red
