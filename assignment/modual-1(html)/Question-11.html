QUE.11 What is the difference between a block-level element and an inline element?

Ans.:-

There are two display values: block and inline.

block-level element:-

A block-level element always starts on a new line.

example:-

Inline element:-

An inline element does not start on a new line.

example:-

Difference Between Inline and Block Elements in HTML:-

  • Block elements always start from a new line. Inline elements never start from a new line. Blockelements cover space from leftto right as far as it can go. Inline elements only cover the space as bounded by the tags in thHTML element.
  • A block element takes up the full width of the content. Unlike inline, there exists a top and bottom margin for theselements. Block-level elements may only appear inside the body tag. Block-level elements create alarger structure thainline elements.
